# 循环结构

循环有三种基本的循环：
 
 - for循环
 - while循环
 - do while循环

这三种循环本质上一样的，可以互相转换。

## for循环

语法如下

```c++
for( s1 : s2 : s3) { 
    block;
}
```
- s1,s2,s3 分别表示 语句1，语句2，语句3
- block 表示多条语句

它的执行过程如下

```plaintext
            +------+
            |  s1  | 执行一次
            +------+
               |
               v
            +------+         +------+  
      +---- |  s2  | <-------|  s3  | <---+
      |     +------+         +------+     |
      |        | YES                      |
      |        v                          |
      |    +--------+                     |
      |    |  block |---------------------+
      |    +--------+
      +-------+
              |
              v
          +-------+
          |  End  |
          +-------+
```

例子1 ：输出1到10，然后输出10到1

```

```
