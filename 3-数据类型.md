# 数字类型

$C++$有以下5种基本数据类型

| 类型     | 关键字 | printf标志 | 标志对应单词 |
|----------|--------|------------|--------------|
| 布尔型   | bool   | %d         | decimal      |
| 字符型   | char   | %c         | char         |
| 整型     | int    | %d         | decimal      |
| 浮点型   | float  | %f         | float        |
| 双浮点型 | double | %lf        | long float   |
